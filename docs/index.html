ngoy

ngoy is a template engine for the JVM, based on the Angular component architecture.

ngoy is a pure Java library with no JavaScript involved.

enjoy ngoy!

Fast & light

Single pass template to byte code compiler.

DOM used only once for compiling.

Streaming output to any destination.

Extensible

Dependency injection, modules, components, directives, pipes, providers/services.

Compile-time hook for components and directives.

Connect other DI system.

Highly compatible

Binding syntax.

[class.*], [style.*], [attr.*]

@HostBinding

*ngFor, *ngIf...

<ng-content>, <ng-container>...

Flexible

Produce any text content.

Use components in plain text formats.

Built-in CLI to render templates/expressions and generate source artifacts.

Render anything from simple text:

Ngoy.renderString("hello {{ name }}", Context.of("name", "world"), System.out);

// hello world

to a full blown app with components, directives, pipes and services:

public class Person {
	public String name;
	public int age;

	public Person(String name, int age) {
		this.name = name;
		this.age = age;
	}
}

@Component(selector = "person", template = "<h1>{{person.name | uppercase}}</h1>age: {{person.age}}")
public class PersonCmp {
	@Input
	public Person person;
}

@Component(selector = "", template = "<html><person *ngFor=\"let it of persons\" [person]=\"it\"></person></html>")
@NgModule(declarations = { PersonCmp.class })
public class App {
	public Person[] persons = new Person[] { 
		new Person("Peter", 22), 
		new Person("Paul", 26), 
		new Person("Mary", 24) };
}

Ngoy.app(App.class)
	.build()
	.render(System.out);

produces:

<html>
	<person><h1>PETER</h1>age: 22</person>
	<person><h1>PAUL</h1>age: 26</person>
	<person><h1>MARY</h1>age: 24</person>
</html>

Distribution

You can download the ngoy binaries from here or via jitpack.io:

build.gradle:

repositories {
	maven { url "https://jitpack.io" }
}

dependencies {
	implementation("com.github.krizzdewizz:ngoy:1.0.0-rc5")
}
