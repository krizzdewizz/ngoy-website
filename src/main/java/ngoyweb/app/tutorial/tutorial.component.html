The Application Shell

Make changes to the application

Change the application title

Open the component class file (AppComponent.java) and change the value of the title field to "Tour of Heroes": 

title = "Tour of Heroes";

Open the component template file (app.component.html) and replace the body element with the following contents: 

<body>
	<h1>{{'{{'}}title{{'}}'}}</h1>
</body>

The double curly braces is an interpolation binding syntax. This interpolation binding presents the component's title property value inside the HTML header tag.

Add application styles

Put your application-wide styles in app.component.css. 

/* Application-wide Styles */
h1 {
  color: #369;
  font-family: Arial, Helvetica, sans-serif;
  font-size: 250%;
}
h2, h3 {
  color: #444;
  font-family: Arial, Helvetica, sans-serif;
  font-weight: lighter;
}
body {
  margin: 2em;
}
body, input[type="text"], button {
  color: #888;
  font-family: Cambria, Georgia;
}
