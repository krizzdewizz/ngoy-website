ngoy ngoy is a template engine for the JVM, based on the Angular+ component architecture. ngoy is a pure Java library with no JavaScript involved.

enjoy ngoy!

Flexible

Produce any text content.

Use components in plain text formats.

Built-in CLI to render templates/expressions and generate source artifacts.

Fast & light

Single pass template to byte code compiler.

DOM used only once for compiling.

Streaming output to any destination.

Extensible

Dependency injection, modules, components, directives, pipes, providers/services.

Compile-time hook for components and directives.

Provide dependencies from any other DI system.
